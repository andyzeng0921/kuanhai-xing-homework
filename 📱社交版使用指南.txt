🍅 番茄王国 - 社交互动版使用指南
========================================

文件：index-social.html

这是专门为你的教学需求定制的版本！

========================================
🎯 核心功能
========================================

✅ 1. 真实照片记录
   - 手机直接拍照
   - 实时上传展示
   - 高清照片保存

✅ 2. 学生互动交流
   - 查看同学的照片
   - 点赞互动
   - 评论讨论

✅ 3. 教师评语
   - 给每张照片写评语
   - 评语单独高亮显示
   - 学生可见教师点评

✅ 4. 观察笔记引导
   - 4个引导性问题
   - 帮助学生有条理地观察
   - 避免乱写

✅ 5. 一键导出
   - 导出所有学生记录
   - 包含照片、笔记、评论
   - 文本格式便于查看


========================================
📱 学生端功能详解
========================================

底部导航栏（4个标签）
-----------------------

1️⃣ 我的番茄 🌱
   - 查看自己拍摄的所有照片
   - 按时间顺序排列
   - 可以看到自己收到的点赞和评论

2️⃣ 拍照 📸
   - 点击调用手机摄像头
   - 拍摄番茄照片
   - 填写观察笔记（有引导问题）
   - 发布到班级

3️⃣ 照片墙 🖼️
   - 看到全班同学的照片
   - 可以点赞别人的照片
   - 可以评论讨论
   - 筛选：全部/最新/热门

4️⃣ 讨论 💬
   - 查看有评论的照片
   - 参与讨论
   - 互相学习


观察笔记引导问题
-----------------------
系统会提示学生回答：

💡 今天番茄有什么变化？
💡 我观察到了什么新发现？
💡 我做了什么照顾工作？
💡 我有什么疑问或想法？

这样学生就不会乱写，而是有针对性地观察！


照片卡片功能
-----------------------
每张照片卡片包含：

- 学生头像和姓名
- 发布时间
- 照片（点击可放大）
- 观察笔记
- 点赞数和评论数
- 点赞按钮
- 评论按钮
- 评论区（可展开/收起）
- 教师评语（黄色高亮显示）


========================================
👨‍🏫 教师端功能详解
========================================

统计面板
-----------------------
显示三个关键数据：
- 📸 总照片数
- 👥 活跃学生数
- 💬 总评论数


学生照片列表
-----------------------
显示所有学生的照片：
- 学生姓名
- 发布时间
- 照片缩略图
- 观察笔记预览
- "写评语"按钮


写评语功能
-----------------------
1. 点击"写评语"按钮
2. 输入评语内容
3. 评语会以黄色背景高亮显示
4. 标注"教师评语"标签
5. 学生可以看到


一键导出功能
-----------------------
点击"💾 一键导出所有记录"

导出内容包括：
- 每个学生的所有照片
- 每张照片的观察笔记
- 点赞数
- 所有评论（包括谁评论的）
- 教师评语

格式：文本文件（.txt）
文件名：番茄王国记录_日期.txt


========================================
💡 使用流程示例
========================================

第1周：建立习惯
-----------------------
1. 学生每天拍照记录
2. 填写观察笔记
3. 发布到班级

教师工作：
- 每天查看学生照片
- 给优秀的照片写评语
- 引导学生关注重点


第2周：互动交流
-----------------------
1. 鼓励学生浏览同学照片
2. 点赞好的观察
3. 在评论区讨论发现

教师工作：
- 组织班级讨论
- "为什么小明的番茄长得快？"
- "大家观察到了什么共同点？"


第4周：总结反思
-----------------------
1. 学生查看自己的成长相册
2. 对比不同时期的照片
3. 总结规律

教师工作：
- 一键导出所有记录
- 整理优秀案例
- 制作班级成长报告


========================================
🎓 教学场景应用
========================================

场景1：课堂展示
-----------------------
1. 教师端打开照片列表
2. 投屏到大屏幕
3. 展示优秀照片
4. 学生分享观察心得


场景2：小组讨论
-----------------------
主题："为什么有的番茄长得快？"

1. 学生在照片墙浏览
2. 对比不同照片
3. 在评论区讨论原因
4. 教师总结规律


场景3：个性化指导
-----------------------
1. 教师查看某学生的所有照片
2. 发现问题（如浇水过多）
3. 写评语提醒
4. 学生看到及时调整


场景4：期末评价
-----------------------
1. 导出所有记录
2. 统计每个学生的：
   - 拍照频率
   - 观察笔记质量
   - 参与讨论情况
3. 给出过程性评价


========================================
⚠️ 重要提示
========================================

数据存储
-----------------------
- 照片存储在浏览器LocalStorage
- 约5MB空间（约50-100张照片）
- ⚠️ 定期导出备份数据
- 清除浏览器数据会丢失


学生账号
-----------------------
当前版本：简化登录
- 学号1-4（示例）
- 实际使用可扩展到更多学生
- 每个学生用自己的设备登录


教师账号
-----------------------
- 选择"教师端"即可进入
- 可以看到所有学生照片
- 可以写评语和导出


照片质量
-----------------------
建议学生：
- 固定角度拍摄
- 光线充足
- 拍摄全景
- 可附带标尺

🚀 快速开始
========================================

方式1：电脑直接用
-----------------------
1. 双击打开：index-social.html
2. 选择"学生端"
3. 输入学号（1-4）
4. 开始使用


方式2：手机扫码用（推荐）
-----------------------
1. 启动服务器.bat
2. 生成二维码.html
   → 修改地址为：index-social.html
   → 生成二维码
3. 手机扫码访问
4. 选择"学生端"
5. 开始拍照记录


教师端使用
-----------------------
1. 打开网站
2. 选择"教师端"
3. 查看学生照片
4. 写评语
5. 导出记录

💡 最佳实践
========================================

给学生的建议
-----------------------
1. 每天固定时间拍照（如早晨8点）
2. 拍摄前先观察1-2分钟
3. 根据引导问题写笔记
4. 浏览同学照片学习
5. 在评论区分享发现


给教师的建议
-----------------------
1. 每天查看学生上传的照片
2. 及时给予评语鼓励
3. 挑选优秀照片课堂展示
4. 引导学生讨论差异
5. 每周导出一次备份


课堂活动设计
-----------------------
活动1：照片对比
- 找出生长最快的番茄
- 分析原因
- 在评论区讨论

活动2：问题解决
- 学生发现问题拍照
- 全班讨论解决方案
- 实施并记录结果

活动3：成长故事
- 每个学生讲述自己的番茄
- 展示照片时间线
- 分享心得体会
