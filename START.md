# 🚀 快速启动指南

## 第一步：安装依赖

在项目目录下运行：

```bash
npm install
```

这将安装所有必需的依赖包（React、Vite、TailwindCSS、Recharts等）。

## 第二步：启动开发服务器

```bash
npm run dev
```

服务器启动后，浏览器访问显示的地址（通常是 http://localhost:5173）

## 🎮 使用指南

### 首页导航
- 点击"我是探秘者"进入学生端
- 点击"我是守护者"进入教师端

### 学生端功能
1. **控制台** - 查看番茄生长状态、每日任务、成就徽章
2. **记录数据** - 使用垂直滑块录入株高，步进器计数叶片/花/果
3. **查看图表** - 生长曲线、数据洞察、历史记录

### 教师端功能
1. **学生列表** - 查看所有学生的记录状态
2. **状态指示灯**：
   - 🟢 绿灯：正常记录
   - 🟡 黄灯：3天未更新（可发送提醒）
   - 🔴 红灯：数据异常
3. **详细档案** - 点击学生查看详情
4. **导出报告** - 生成电子档案袋

## 📊 默认数据

系统内置了2个示例学生：
- **小明**：定植15天，有完整记录
- **小红**：定植12天，记录较少

你可以切换学生账号查看不同的数据（修改localStorage中的currentUser.id）

## 🔧 开发工具

- 按 `F12` 打开浏览器开发者工具
- 在 Console 中可以查看localStorage数据：
  ```javascript
  localStorage.getItem('students')
  localStorage.getItem('currentUser')
  ```

## 💡 提示

- 数据保存在浏览器LocalStorage中
- 清除浏览器数据会重置为默认学生数据
- 建议使用Chrome或Edge浏览器以获得最佳体验
