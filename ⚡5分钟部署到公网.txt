🎯 最简单方式（推荐新手）
========================================

使用 Vercel - 完全免费，5分钟搞定！

第1步：注册Vercel
-----------------------
1. 打开浏览器，访问：https://vercel.com
2. 点击右上角"Sign Up"
3. 选择"Continue with GitHub"
4. 用你的GitHub账号登录（没有就注册一个）


第2步：部署网站
-----------------------
1. 登录后点击"Add New..." → "Project"
2. 选择"Continue with GitHub"
3. 授权Vercel访问你的GitHub
4. 或者直接点击"Import Third-Party Git Repository"

简单方式（推荐）：
  1. 点击页面底部的"Browse"
  2. 选择你的文件夹：C:\Users\MECHREVO\Desktop\web homework
  3. 点击"Upload"
  4. 等待30秒
  5. 完成！


第3步：获得网址
-----------------------
部署完成后，你会看到：

🎉 Congratulations!
Your project is live at: https://xxx.vercel.app

点击这个网址，就能访问你的番茄王国了！


第4步：绑定你的域名（andyzeng.xin）
-----------------------
1. 在Vercel项目页面，点击"Settings"
2. 点击左侧"Domains"
3. 在输入框输入：andyzeng.xin
4. 点击"Add"

5. Vercel会提示你添加DNS记录
   复制显示的记录值（类似：cname.vercel-dns.com）

6. 打开阿里云控制台：https://dns.console.aliyun.com
7. 找到 andyzeng.xin，点击"解析设置"
8. 点击"添加记录"：
   - 记录类型：CNAME
   - 主机记录：@
   - 记录值：粘贴Vercel给的值
   - TTL：10分钟
   - 点击"确定"

9. 再添加一条www记录：
   - 记录类型：CNAME
   - 主机记录：www
   - 记录值：同上
   - 点击"确定"

10. 等待5-10分钟生效

11. 访问：https://andyzeng.xin
    完成！🎉

========================================
🔧 如果遇到问题
========================================

问题1：GitHub账号不知道怎么注册
解决：
  1. 访问：https://github.com
  2. 点击"Sign up"
  3. 输入邮箱、密码
  4. 完成验证
  5. 返回Vercel继续

问题2：DNS解析不知道怎么改
解决：
  1. 登录阿里云：https://www.aliyun.com
  2. 控制台 → 域名 → andyzeng.xin
  3. 点击"解析设置"
  4. 按上面的步骤添加CNAME记录

问题3：等了很久还是访问不了
解决：
  1. DNS生效需要时间（5分钟-2小时）
  2. 可以先用Vercel给的临时网址测试
  3. 清除浏览器缓存再试
  4. 用手机流量访问试试（排除DNS缓存）

问题4：提示需要修改DNS服务器
解决：
  你的域名DNS服务器当前是Cloudflare的
  需要改回阿里云DNS：
  
  1. 阿里云控制台 → 域名
  2. 找到 andyzeng.xin
  3. 点击"DNS修改"
  4. 修改为：
     dns9.hichina.com
     dns10.hichina.com
  5. 提交，等待24-48小时生效
