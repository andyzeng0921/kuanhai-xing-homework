# 🍅 番茄王国的自然笔记

一个温馨的教育平台，用于记录学生种植番茄的成长过程，培养科学观察和数据分析能力。

> ⭐ **最新版本**：社交互动版 - 支持照片分享、评论讨论、教师评语、一键导出！

## 📌 三个版本对比

| 版本 | 文件 | 功能特点 | 推荐场景 |
|------|------|---------|---------|
| 基础版 | `index-standalone.html` | 数据录入、生长曲线、图表分析 | 快速数据记录 |
| 增强版 | `index-mobile-enhanced.html` | 基础版 + 手机拍照上传 + 过程性评价报告 | 个人成长档案 |
| **社交版** ⭐ | `index-social.html` | 真实照片 + 学生互动 + 评论讨论 + 教师评语 | **班级集体使用** |

## ✨ 特色功能

### 设计系统
- 🎨 **番茄王国主题**：番茄红(#FF6347) + 嫩芽绿(#4CAF50) + 米黄纸质背景(#FEFBF5)
- 🔤 **圆角字体**：温馨友好的视觉体验
- 🍬 **糖果按钮**：有趣的交互动画

### 社交版核心功能（最新推荐）

#### 学生端
1. **🌱 我的番茄**
   - 查看自己拍摄的所有照片
   - 照片按时间排列
   - 查看收到的点赞和评论

2. **📸 拍照记录**
   - 手机相机直接拍摄
   - 观察笔记引导（4个问题提示）
   - 一键发布到班级

3. **🖼️ 班级照片墙**
   - 浏览全班同学的照片
   - 点赞互动
   - 评论讨论
   - 筛选：全部/最新/热门

4. **💬 讨论区**
   - 参与热门讨论
   - 学习他人观察
   - 互动交流

#### 教师端
1. **统计面板**
   - 总照片数统计
   - 活跃学生数
   - 总评论数

2. **评语管理**
   - 给每张照片写评语
   - 评语高亮显示
   - 实时反馈指导

3. **一键导出**
   - 导出所有学生记录
   - 包含照片信息、观察笔记
   - 包含所有评论和教师评语
   - 文本格式便于查看

### 基础版/增强版功能
1. **探秘者控制台**
   - 个人信息展示（头像、等级、勋章）
   - 实时番茄生长动画
   - 生长阶段指示器（幼苗期/成长期/开花期/结果期）
   - 每日任务提醒
   - 成就徽章系统

2. **生长记录实验室**
   - 垂直滑块录入株高（防止错误数据）
   - 步进器计数叶片/花朵/果实
   - 探究性提问（预测-观察-反馈）
   - 照片上传功能
   - 观察笔记记录

3. **数据可视化大厅**
   - 生长曲线图表
   - 全班平均对比功能
   - 智能数据洞察（最快生长期、重要里程碑）
   - 历史记录表格

### 教师端功能
1. **全景监控舱**
   - 班级数据概览（学生总数、活跃度、平均株高）
   - 智能状态指示灯
     - 🟢 绿灯：正常记录
     - 🟡 黄灯：超过3天未更新
     - 🔴 红灯：数据逻辑错误
   - 学生详细档案查看
   - 一键提醒功能
   - 电子档案袋导出（PDF/JSON）

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📦 技术栈

- **前端框架**：React 18 + Vite
- **样式方案**：TailwindCSS
- **路由管理**：React Router v6
- **图表库**：Recharts
- **图标库**：Lucide React
- **状态管理**：LocalStorage + React Hooks

## 🎯 教育价值

1. **科学探究能力**：通过预测-观察-记录-分析的流程培养科学思维
2. **数据素养**：学会用图表和数据讲述故事
3. **责任感培养**：每日记录培养学生的坚持和责任心
4. **成就激励**：徽章系统提供正向反馈
5. **循证学习**：基于数据的反思和讨论

## 🎨 设计理念

整个网站以"自然笔记本"为隐喻：
- 米黄色背景模拟纸质笔记本
- 圆角设计营造温馨感
- 番茄和植物元素贯穿始终
- 游戏化元素提升参与度

## 📝 默认数据

系统内置了2个示例学生数据，方便体验：
- 小明（定植15天，5个勋章）
- 小红（定植12天，1个勋章）

## 🔒 数据存储

使用浏览器LocalStorage进行数据持久化，适合教学演示和单机使用。
