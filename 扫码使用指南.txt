🍅 番茄王国 - 手机扫码访问指南
========================================

📱 完整操作步骤（3步搞定！）
========================================

第1步：启动本地服务器
-----------------------
方法A（推荐）：
  1. 双击运行：启动服务器.bat
  2. 看到命令行窗口显示"Serving HTTP on..."即成功
  3. 保持窗口打开，不要关闭！

方法B（备选）：
  1. 在当前文件夹打开PowerShell或CMD
  2. 输入：python -m http.server 8080
  3. 按回车，保持窗口打开

⚠️ 如果提示"python不是内部或外部命令"：
   说明电脑没装Python，请使用方法C

方法C（无需Python）：
  1. 下载HFS软件（HTTP File Server）
  2. 或使用任意本地服务器工具
  3. 将端口设置为8080


第2步：生成二维码
-----------------------
  1. 双击打开：生成二维码.html
  2. 点击"自动检测IP地址"按钮
  3. 选择一个IP地址（通常选192.168开头的）
  4. 点击"生成二维码"按钮
  5. 保存或截图二维码

💡 如果自动检测失败：
   1. 点击"手动输入"
   2. 按Win+R，输入cmd，回车
   3. 输入：ipconfig
   4. 找到"IPv4 地址"，例如：192.168.1.100
   5. 将IP地址输入到网页中


第3步：手机扫码访问
-----------------------
  1. 确保手机和电脑连接同一个WiFi！！！
  2. 打开微信或手机浏览器的扫一扫
  3. 扫描生成的二维码
  4. 自动打开番茄王国网站


========================================
🔥 常见问题解决
========================================

Q1: 扫码后无法访问？
A1: 
  ✅ 检查手机和电脑是否连同一WiFi
  ✅ 检查Windows防火墙，允许端口8080
  ✅ 尝试关闭防火墙测试
  ✅ 确保服务器还在运行（命令行窗口没关）

Q2: 找不到Python怎么办？
A2: 
  ✅ 方案1：直接双击index-standalone.html在电脑浏览器打开
  ✅ 方案2：下载安装Python（python.org）
  ✅ 方案3：使用其他简单HTTP服务器软件

Q3: 生成的二维码无法显示？
A3:
  ✅ 检查是否联网（生成二维码需要联网一次）
  ✅ 或者手动在手机浏览器输入网址
  ✅ 网址格式：http://你的IP:8080/index-standalone.html

Q4: 不同WiFi能访问吗？
A4:
  ❌ 不能！必须同一WiFi
  ✅ 如需跨网络访问，需要使用内网穿透工具


========================================
💻 查看电脑IP地址的方法
========================================

Windows系统：
  1. 按 Win + R
  2. 输入：cmd
  3. 输入：ipconfig
  4. 查找"无线局域网适配器 WLAN"下的"IPv4 地址"

Mac系统：
  1. 系统偏好设置 → 网络
  2. 选择WiFi
  3. 查看IP地址

Linux系统：
  终端输入：ip addr 或 ifconfig


========================================
🌐 其他访问方式（不用二维码）
========================================

方式1：手机直接输入网址
  http://你的IP:8080/index-standalone.html
  例如：http://192.168.1.100:8080/index-standalone.html

方式2：电脑浏览器访问
  http://localhost:8080/index-standalone.html
  或
  http://127.0.0.1:8080/index-standalone.html
